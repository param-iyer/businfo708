Open Box Calculus Optimization

The Dynamic Symmetric Problem

When we cut four identical square corners of size $x \times x$ from a cardboard sheet of dimensions $W \times L$, the resulting folded box has dimensions:

  • Height: $x$
  • Width of base: $W - 2x$
  • Length of base: $L - 2x$

The volume function to optimize is:

$$V(x) = x(W - 2x)(L - 2x) = 4x^3 - 2(W+L)x^2 + WLx$$

Finding the Peak Using Derivatives

To locate the exact mathematical point of maximum volume, we find the first derivative $V'(x)$ and solve for $V'(x) = 0$:

$$V'(x) = 12x^2 - 4(W+L)x + WL = 0$$

Using the quadratic formula to find the roots:

$$x = \frac{4(W+L) \pm \sqrt{16(W+L)^2 - 48WL}}{24}$$
